# Project Task List: TTRPG Story Web App for GitHub Pages

## Overview

This web app will display stories from a tabletop roleplaying game, with markdown-based storylines, dynamic character sheets, and campaign navigation. It will be deployable on GitHub Pages.

## Task List

### 1. Project Setup

- [ ] Scaffold Next.js app for static export
- [ ] Configure for GitHub Pages deployment
- [ ] Add Tailwind CSS for styling

### 2. Core Features

- [ ] Home page with overview
- [ ] Navigation bar for campaign selection
- [ ] Markdown rendering for storylines
- [ ] Collapsible sections for rules and HRP
- [ ] Dynamic character sheets
- [ ] Character pages with story and sheet
- [ ] Internal linking from story to character pages

### 3. Content Management

- [ ] Organize stories and campaigns in markdown files
- [ ] Organize character data and stories

### 4. Deployment

- [ ] Test static export
- [ ] Deploy to GitHub Pages
- [ ] Document deployment steps

### 5. Polish & QA

- [ ] Responsive design
- [ ] Accessibility checks
- [ ] Final content review
